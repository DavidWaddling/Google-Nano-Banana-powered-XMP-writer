# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/drive/1iyX66LigT22S2oysL9wzOq1k0C-YtpFJ

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Configure your API Key:
   - Open the `services/apiKey.ts` file in your editor.
   - Replace the placeholder text `"[Put your Google Gemini API key Here]"` with your actual Google Gemini API key.
   - You can get a key from [Google AI Studio](https://makersuite.google.com/app/apikey).
3. Run the app:
   `npm run dev`
